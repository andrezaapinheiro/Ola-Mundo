Curso de Git e Github

Foi lecionado as definições de Git e Github. Nele foi apresentado a criação de cada um seus criadores.

Git

Git é um sistema de controle de versão de códigos. Nele é possível, através do gerenciamento de ramificações, que várias pessoas de um time trabalhem em paralelo e integrem ao programa mantendo a ordem de todas as partes do programa. E também é um sistema que pode controlar as mudanças para evitar erros.

Github

GitHub é uma plataforma feita para hospedagem de códigos de programação. Os objetivos principais são o controle e a colaboração, permitindo que usuários desenvolvam projetos juntos, estando em qualquer lugar do mundo.
